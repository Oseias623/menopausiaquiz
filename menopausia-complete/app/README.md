
# App Menopausia Con Claridad

Aplicação oficial do Programa Antiinflamatório para a Menopausa.

## Funcionalidades
- Gamificação (Checklist Diário)
- Quiz de Personalidade (Vitalidade, Desinflamação, Sincronização)
- Acesso a Conteúdos Exclusivos e Bônus
- Integração com Hotmart

## Tecnologias
- React + Vite
- TailwindCSS
- Supabase
